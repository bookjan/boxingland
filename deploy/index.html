教練

楊本聖

課程包含基礎到中級技術，除空拳外，使用器械（沙包、藥球）及手靶輔助教學，沒有對打。

基礎課程包含：

  1. 直拳
  2. 勾拳
  3. 上勾拳
  4. 撥擋閃

初級課程，將上述基本動作串成組合拳連打：

  1. 直拳勾拳上勾拳基本組合。
  2. 直拳勾拳上勾拳＋撥拳反擊組合。
  3. 直拳勾拳上勾拳＋擋拳反擊組合。
  4. 直拳勾拳上勾拳＋閃躲反擊組合。
  5. 綜上，對手隨機出拳，要能達到反射式的防禦加反擊組合。

中級課程，將上述組合再搭配腳步，構成各距離角度的行進間連打：

  1. 左右跳閃＋切角度的組合連擊。
  2. 跟著對手、滾動行進＋切角度的組合連擊。
  3. 跟著對手、閃躲＋滾動行進＋切角度的組合連擊。

教練

張語彤

大家好，我是語彤。一名專注於拳擊體能與技術的教練。

我深信拳擊不僅是擊打，更是一場關於核心控制、身體協調與心理韌性的自我對話。無論你是想透過拳擊釋放壓力、雕塑體態，還是希望為孩子建立專注力與自信心，我都能為你量身打造訓練課表。

針對團體課程：擅長營造高能量的訓練氛圍，讓大家在互動中激發潛力，既能爆汗減脂，又能學習到紮實的防衛技巧。

針對私教課程：精準分析你的身體狀態，將核心穩定融入拳擊動作，幫你改善體態並提升發力效率，確保動作精準且安全。

針對兒童拳擊：採取啟發式的教學，將枯燥的訓練轉化為趣味挑戰，強調肢體協調與專注力培養，讓孩子在運動中學會尊重與自我保護。

FAQ

課程與設備常見問題

想上課或想詢問設備，都可以先從這裡確認。

完全沒基礎可以上拳擊課嗎？

可以。入門課會從站姿、步伐、出拳、防守與體能負荷開始調整。

設備商品可以直接下單嗎？

目前不做線上結帳。請先傳場地尺寸、用途與預算，我們協助確認規格。

可以協助規劃拳擊教室嗎？

可以。可討論拳擊台、沙袋動線、地墊、牆墊、柱墊與基礎訓練器材。

品牌型錄裡的商品都能詢問嗎？

CROSSMAX、GYMXF、GYMPAC 等型錄商品可依分類詢問，實際規格以報價確認。
